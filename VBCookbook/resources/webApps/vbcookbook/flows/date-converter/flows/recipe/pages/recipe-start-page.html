Summary:

  1. Create a page variable to hold the converter options.
  2. Create an Input Date component with setting the converter to the page variable.
  3. Create combobox to choose desired format.
  4. Add an on-value-changed event to update the pattern property of the page variable.
Create the Page Model
  1. Add a dateFormat page variable of type Object. Give it the following structure: 
    {
  "type": "",
  "options": {
    "pattern": ""
  }   
}
  2. Give dateFormat the default value of: 
    {
  "type": "datetime",
  "options": {
    "pattern": "MM/dd/yy"
  }   
}
 Create the UI
  1. Drop the Form Layout, Date Picker, and Select One components on the page.
  2. Give the Select One the following options:
    • yy-MM-dd
    • MM/dd/yy
  3. Set the converter property of the Date Picker to [[ $variables.dateFormat ]]
 Add the Event and Action Chain
  1. Select the Select One component and use the Properties pane to add an event listeren for value.
  2. Drag an Assign Variables action to the page. Use it to assign $chain.variables.value to $page.variables.dateFormat.options.pattern
