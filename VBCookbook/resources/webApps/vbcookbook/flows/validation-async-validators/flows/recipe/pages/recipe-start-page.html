Step 1: Create Form with Input Text element
  1. Open design view
  2. Dnd Input Text element on to page and set following properties using PI.
    • "validators" property to "[[$variables.validators]]". This is JET numberRange validator with min=100.
    • "converter" property to a currency converter as shown.
    • "value" property to "{{$variables.quantityLimit}}" to capture the entered value.
    • "async-validators" to "[[[$page.functions.asyncValidator]]]", a variables defined on PageModule. This is an object that duck-types AsyncValidator. Its API is validate(): Promise and hint: Promise. Note: Async validator type is not supported as variables in Oracle Visual Builder so have to use "$page.functions.*" .
  3. Type in 10001, you should see the async-validators error message.
  4. Type in 10000, you should see the async-validators error message disappear and the valid state is valid.
  5. Type in 99, you should see the validators error message.
  6. Clear field and tab away, you should see the required error message.
  7. Type in 999, you should see no error messages and valid state should be valid.
